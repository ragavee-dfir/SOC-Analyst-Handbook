# SOC Analyst Handbook

## About

This repository contains my personal SOC Analyst study notes created while learning cybersecurity, digital forensics, Windows internals, and incident response.

The purpose of this handbook is to document the tools, concepts, and investigation techniques commonly used by SOC analysts. Each chapter summarizes the purpose of the tool, how to use it, what to investigate, common red flags, and lessons learned from hands-on practice.

## Topics Covered

- Process Explorer
- Event Viewer
- Sysmon
- Autoruns
- TCPView
- Process Monitor (Procmon)
- RAMMap
- Windows Defender
- CurrPorts
- Windows Internals
- Networking Fundamentals
- Active Directory
- PowerShell
- Splunk
- Threat Hunting
- SOC Investigation Workflow

## Goal

Build a practical reference guide that supports continuous learning and can be used during labs, interview preparation, and real-world SOC investigations.
